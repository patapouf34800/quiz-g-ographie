🎯 But du jeu

Géo Quiz est un jeu éducatif pour apprendre la géographie mondiale ! Ton objectif est de découvrir et mémoriser les 195 pays du monde.

🎮 Comment jouer

  • Choisis un mode : Nomme les pays, Capitales ou Expert (pays + capitales)
  • Choisis une zone : Monde, Europe, Asie, Afrique, Amérique ou Océanie
  • Lance le quiz : On te montre le drapeau et la silhouette d'un pays
  • Trouve la réponse : Tape le nom du pays et/ou sa capitale selon le mode
  • Collecte les pays : Chaque pays trouvé est ajouté à ta collection
  • Débloque des badges : Atteins des objectifs pour gagner des récompenses

🌟 Système de progression

Cadres dorés (maîtrise des pays)

Obtiens un cadre doré autour du drapeau en maîtrisant le nom du pays :

  • Condition : Valide le pays 6+ fois en mode "Nomme les pays" et/ou "Expert"
  • Progression : (Mode Nomme + Mode Expert) ≥ 6
  • Exemple : 4x en Nomme + 2x en Expert = Cadre doré ✅

Rubans (maîtrise des capitales)

Gagne des rubans en maîtrisant les capitales :

  • 🥉 Ruban Bronze : 1-5 validations en mode "Capitales" et/ou "Expert"
  • 🥈 Ruban Argent : 6+ validations en mode "Capitales" et/ou "Expert" (mais moins de 6 en Expert seul)
  • 🥇 Ruban Or : 6+ validations en mode "Expert" uniquement

💡 Astuce : Le mode Expert compte à la fois pour les cadres dorés ET les rubans !

🏆 Système de badges

🗺️ Badges Exploration

Obtiens un cadre doré sur chaque pays d'une zone pour débloquer ces badges.

  • 🔒 0-24% : Badge verrouillé
  • 🥉 25-49% : Médaille de bronze
  • 🥈 50-74% : Médaille d'argent
  • 🥇 75-99% : Médaille d'or
  • ✨ 100% : Badge complet !

🏛️ Badges Capitales

Obtiens un ruban argent ou or sur chaque pays d'une zone pour débloquer ces badges.

Même système de progression (Bronze/Argent/Or/Complet) que les badges Exploration.

💡 Astuces

  • Commence par l'Europe si tu débutes, c'est plus facile !
  • Observe bien les drapeaux, ils donnent souvent des indices
  • Retourne régulièrement dans ta collection pour réviser
  • Le mode Expert est plus difficile mais fait progresser plus vite !
  • Essaie de compléter un continent entier pour devenir expert !

🌍 Géographie

Le jeu inclut les 195 pays membres de l'ONU, répartis sur 5 continents :

  • Europe : 49 pays
  • Asie : 43 pays
  • Afrique : 54 pays
  • Amérique : 35 pays
  • Océanie : 14 pays
